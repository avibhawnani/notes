# Blogster : A live blog publishing site

- This project includes following features :
  - SignUp / Login
  - Add Post
  - Edit Post
  -  Get Posts
  -  Active / Inactive posts
  -  Delete Posts
  -  Logout

- This project includes `React` , `Tailwind` , `react-hook-forms` , `redux-toolkit` , `react-redux` , `vite` , `tinymce-react` , `html-react-parser`
